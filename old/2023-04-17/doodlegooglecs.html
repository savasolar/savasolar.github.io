Doodle Google


Web Design / Development • Visit Website

What is This?



This is a recent recreation of a project I made a few years ago, when I first encountered Google’s Programmable Search Engine tool. Since Google gave developers the option to integrate their search tools into custom web designs, it felt like it would be fun to make a fully functional search engine.


I also loved the idea of scanning drawings I had made by hand and breathing life into them with code.



The Design and Development Process



First things first, I had to look through the Google Search page and recreate every visual element with pen and paper. This took quite a while:











Then, I started arranging each element on a webpage, largely relying on CSS to position everything properly.


One of the biggest challenges was making a scalable HTML element out of a static hand-drawn image. One such example is the search bar, which changes its width depending on the user’s display size.


After creating the web layout, it was also necessary to reposition, resize, and otherwise readjust most of the elements on the site for mobile displays.






The Main Takeaway



This project was simply done for the pleasure of bringing a cool idea to life. Its big takeaway is simply that web design can be fun and as creative as you want it to be!
